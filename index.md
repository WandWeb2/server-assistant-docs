# Server Assistant

A multi-tenant Discord moderation and automation bot. Each server gets its own isolated configuration — warnings, notes, AutoMod rules, and settings stay scoped to that guild.

## Add to your Discord server

Coming soon to the [Discord App Directory](https://discord.com/application-directory) and major bot listings.

## Features at a glance

- 🛡️ Role-based moderation with custom tier support
- 🤖 AutoMod engine with curated filter packs and configurable escalation
- 🚨 Anti-raid detection with adjustable sensitivity
- 🔐 DM-button verification gate
- ⚠️ Persistent warning system with DM notifications
- 📝 Private staff notes invisible to users
- 🎨 AI image generation with free fallback (no key required)
- 🧠 AI-assisted moderation reports via right-click
- 👋 Auto-onboarding DMs with placeholder substitution
- 📋 Custom canned response snippets
- 📅 Scheduled reminders with timezone awareness
- 🔑 Encrypted secrets vault for all API keys
- 📊 Weekly activity digests per server
- ⚡ One-click presets (AutoMod, ladder, onboarding template, anti-raid)
- 🎨 Per-server branding (embed colour, bot nickname, default timezone)
