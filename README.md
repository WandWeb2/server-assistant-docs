# Server Assistant — Public Documentation

This repository hosts the public-facing documentation (privacy policy, terms of service, landing page) for the **Server Assistant** Discord bot.

The bot's source code lives in a separate private repository.

## Live site

📄 https://wandweb2.github.io/server-assistant-docs/

- [Privacy Policy](https://wandweb2.github.io/server-assistant-docs/privacy/)

## Editing

This site is a [Jekyll](https://jekyllrb.com/) project hosted on [GitHub Pages](https://pages.github.com/). To update content, edit the `.md` files in this repo and push — Pages rebuilds automatically within a minute.
